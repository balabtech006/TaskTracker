# Installation Method

Install Node Server

Step 1 : cd server
Step 2 : NPM install
Step 3 : node server.js

Install Frontend dependencies

Step 1 : cd client
Step 2 : NPM install
Step 3 : NPM start

MongoDB Setup

DB name : timeSheet

start mongoDB in shell or use compass to start mongoDB cnnection
